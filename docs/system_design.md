# AIdeal CPS 系统设计文档

## 1. 项目概述
- **项目名称**：AIdeal CPS（智省优选）
- **目标**：基于京东联盟与微信服务号，开发一个AI导购与联盟返佣系统，提供个性化商品推荐与推广链接生成功能。
- **技术栈**：
  - 后端：FastAPI, MySQL, Alembic
  - 部署：阿里云 ECS（Ubuntu），systemd + Nginx
  - 前端：H5 页面作为服务号承接页
  - AI：Qwen（DashScope）

## 2. 系统架构
- **后端架构**：
  - **FastAPI** 作为 API 框架，提供高效、可扩展的接口。
  - **MySQL** 用于存储商品信息、用户数据、推荐数据等。
  - **Alembic** 用于数据库迁移，保证数据结构与业务需求一致。
- **前端架构**：
  - **微信服务号** 用于用户交互，通过 H5 页面展示商品推荐和推广链接。
- **AI推荐系统**：
  - 基于用户历史数据与商品特性，采用简化的评分模型进行商品排序和推荐。

## 3. 完成的功能模块
- **CPS中台开发**：已完成CPS中台的基础功能，能够通过京东联盟API生成推广链接。
- **H5页面开发**：实现了H5页面作为微信服务号的承接页，提供用户互动功能。
- **API接口**：已完成与京东联盟API的基本集成，包括生成推广链接的功能。
- **推荐引擎**：初步实现了商品推荐系统，基于销量、佣金等数据对商品进行排序。

## 4. 系统部署
- **部署平台**：阿里云 ECS（Ubuntu）
- **服务部署**：通过 `systemd` 管理服务启动与停止，使用 Nginx 作为反向代理。

## 5. 开发与协作习惯
- **代码管理**：使用 GitHub 进行版本控制，所有开发内容通过提交 commit 进行版本管理。
- **操作规范**：所有操作必须是 `copy-paste` 可执行，不允许手动修改文件。
- **持续集成**：每次系统变动后，更新设计文档，并通过 GitHub 进行版本更新。

## 6. 未来发展与改动
- **微信服务号集成**：进一步集成微信服务号，确保用户可以通过微信获取推荐商品并生成推广链接。
- **AI推荐算法优化**：基于更多维度（如用户画像、商品折扣等）优化推荐算法，提高用户转化率。
- **API接口优化**：解决当前接口权限问题，确保能更稳定地生成推广链接。
- **数据存储与性能优化**：随着用户与商品数据量的增加，进行数据库与系统性能优化。
